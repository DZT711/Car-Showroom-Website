## Enhancement Rules

Do NOT replace existing polished sections unless necessary.

Prefer enhancing current implementations.

Maintain the existing:

- Typography
- Color palette
- Motion language
- Layout rhythm
- Luxury branding

Every new feature should feel like a natural extension of the current website rather than a separate design system.

All enhancements must preserve performance, responsiveness, accessibility, and maintainability.
